## Project Description
This project is based on Open Source Software concepts and Linux shell scripting. It includes scripts to analyze system information, verify software installation, audit directories, and process log files.



## Scripts

1. script1.sh  
   Displays system information such as kernel version, user, uptime, date, and Git version.

2. script2.sh  
   Checks whether Git is installed on the system.

3. script3.sh  
   Audits important directories and shows their permissions.

4. script4.sh  
   Analyzes system log files and counts error occurrences.

5. script5.sh  
   Generates an open source manifesto using user input.
